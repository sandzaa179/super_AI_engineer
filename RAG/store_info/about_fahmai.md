# เกี่ยวกับฟ้าใหม่ (About FahMai)

วันที่อัปเดต: 1 มีนาคม 2569

---

## จุดเริ่มต้นของฟ้าใหม่

ฟ้าใหม่ (FahMai) ก่อตั้งขึ้นในปี 2558 โดย คุณสมชาย ฟ้าสว่าง ด้วยความเชื่อมั่นว่าคนไทยทุกคนสมควรเข้าถึงเทคโนโลยีที่ดีได้ในราคาที่เป็นธรรม และได้รับประสบการณ์การซื้อสินค้าออนไลน์ที่ไว้วางใจได้ เขาเริ่มต้นจากร้านออนไลน์เล็กๆ ที่จำหน่ายสมาร์ทโฟนและอุปกรณ์เสริมไม่กี่รายการ ก่อนจะค่อยๆ เติบโตจนกลายเป็นแพลตฟอร์มอีคอมเมิร์ซด้านอิเล็กทรอนิกส์ที่ครบวงจรที่สุดแห่งหนึ่งในประเทศไทย

ในช่วงแรก คุณสมชายมุ่งเน้นไปที่การสร้างความไว้วางใจกับผู้ซื้อ ด้วยการให้ข้อมูลสินค้าที่โปร่งใส มีระบบการคืนสินค้าที่ชัดเจน และบริการหลังการขายที่จริงจัง แนวทางนี้สร้างฐานลูกค้าประจำให้เติบโตอย่างรวดเร็ว และในปี 2561 ฟ้าใหม่ได้เปิดตัวแบรนด์สินค้าในเครือแบรนด์แรกอย่างเป็นทางการ

ปัจจุบัน ฟ้าใหม่มีพนักงานมากกว่า 200 คน มีศูนย์บริการทั่วประเทศ 5 แห่ง ได้แก่ กรุงเทพมหานคร 3 แห่ง เชียงใหม่ 1 แห่ง และภูเก็ต 1 แห่ง พร้อมฐานสมาชิกที่ใช้งานจริงมากกว่า 500,000 ราย

## วิสัยทัศน์และพันธกิจ

**วิสัยทัศน์:** เป็นแพลตฟอร์มอิเล็กทรอนิกส์ที่คนไทยไว้วางใจมากที่สุด ด้วยสินค้าที่หลากหลาย คุณภาพที่ตรวจสอบได้ และบริการหลังการขายที่เหนือระดับ

**พันธกิจ:**
- นำเสนอสินค้าเทคโนโลยีคุณภาพสูงในราคาที่เข้าถึงได้
- สร้างประสบการณ์การช้อปปิ้งออนไลน์ที่ปลอดภัยและโปร่งใส
- พัฒนาสินค้าภายใต้แบรนด์ในเครือที่ตอบโจทย์ผู้ใช้ไทยโดยเฉพาะ
- สนับสนุนพันธมิตรแบรนด์ที่มีคุณภาพให้เข้าถึงตลาดไทยได้ง่ายขึ้น
- ดูแลลูกค้าอย่างต่อเนื่องผ่านศูนย์บริการและระบบสมาชิก

## แบรนด์ในเครือฟ้าใหม่ (House Brands)

ฟ้าใหม่พัฒนาสินค้าภายใต้แบรนด์ตัวเองทั้งหมด 5 แบรนด์ แต่ละแบรนด์มีจุดเด่นและกลุ่มเป้าหมายที่แตกต่างกันชัดเจน

### สายฟ้า (SaiFah) — สมาร์ทโฟนและแท็บเล็ต

สายฟ้าเป็นแบรนด์ที่เน้นอุปกรณ์พกพาอัจฉริยะ ครอบคลุมตั้งแต่สมาร์ทโฟนระดับเริ่มต้นสำหรับผู้ใช้ทั่วไปไปจนถึงแฟล็กชิปสเปคสูงสำหรับคนที่ต้องการประสิทธิภาพสูงสุด รวมถึงแท็บเล็ตหลายรุ่นที่ออกแบบมาเพื่อการทำงาน การเรียน และความบันเทิง

สายฟ้ามีจุดยืนที่ชัดเจนในการออกแบบที่ตอบโจทย์ผู้ใช้ภาษาไทย ทั้งในแง่ของ UI ระบบปฏิบัติการ FahMai OS และฟีเจอร์ที่สอดคล้องกับพฤติกรรมการใช้งานของคนไทย ไม่ว่าจะเป็นการรองรับฟอนต์ไทย, การชำระเงินผ่าน QR Code หรือความเข้ากันได้กับซิมการ์ดทุกค่ายในประเทศไทย

สินค้าในกลุ่มสายฟ้าครอบคลุมหลายซีรีส์ ตั้งแต่ซีรีส์ที่เน้นความคุ้มค่าสำหรับงบประมาณจำกัด ไปจนถึงซีรีส์ระดับสูงที่มาพร้อมกับเทคโนโลยีกล้องและหน้าจอล้ำสมัย รวมถึงแท็บเล็ตรุ่นพิเศษสำหรับกลุ่มเฉพาะทาง เช่น เด็ก ผู้สูงอายุ หรือนักออกแบบ

### ดาวเหนือ (DaoNuea) — คอมพิวเตอร์และแล็ปท็อป

ดาวเหนือเป็นแบรนด์สำหรับผู้ที่ต้องการเครื่องมือการทำงานและความบันเทิงในบ้านที่ทรงพลัง ครอบคลุมแล็ปท็อปหลายประเภทตามกลุ่มผู้ใช้ ไม่ว่าจะเป็นนักเรียนนักศึกษา นักสร้างคอนเทนต์ โปรแกรมเมอร์ หรือเกมเมอร์ รวมถึงคอมพิวเตอร์ตั้งโต๊ะและ All-in-One สำหรับใช้งานภายในบ้านและสำนักงาน

ดาวเหนือเป็นแบรนด์ที่ให้ความสำคัญกับความสมดุลระหว่างประสิทธิภาพและราคา โดยใช้ชิปเซ็ตและ GPU จากผู้ผลิตชั้นนำเพื่อให้มั่นใจในคุณภาพ พร้อมทั้งการรับประกันที่ครอบคลุมรวมถึงบริการ On-site service สำหรับแล็ปท็อปในปีแรก

### คลื่นเสียง (KluenSiang) — เสียงและมัลติมีเดีย

คลื่นเสียงเชี่ยวชาญด้านอุปกรณ์เสียงและความบันเทิง ประกอบด้วยหูฟังครอบหูหลายรุ่นตั้งแต่ระดับเริ่มต้นจนถึงระดับ Hi-Fi, หูฟัง True Wireless (TWS) ที่รองรับ ANC, ลำโพงพกพา, ลำโพงบลูทูธ และระบบเสียง Soundbar สำหรับห้องนั่งเล่น

แบรนด์คลื่นเสียงเกิดจากความเชื่อว่าประสบการณ์ด้านเสียงที่ดีไม่ควรเป็นสิทธิพิเศษของคนกลุ่มใดกลุ่มหนึ่ง จึงมีสินค้าในราคาที่หลากหลาย โดยยังคงรักษาคุณภาพเสียงที่สมกับราคาในแต่ละระดับ

### วงโคจร (WongKhojorn) — Wearables และฟิตเนส

วงโคจรเป็นแบรนด์ด้านอุปกรณ์สวมใส่ (Wearable) ที่ผสมผสานเทคโนโลยีสุขภาพและการออกกำลังกายเข้ากับการออกแบบที่ใช้งานได้จริงในชีวิตประจำวัน ผลิตภัณฑ์ครอบคลุมสมาร์ทวอทช์หลายระดับและฟิตเนสแทร็กเกอร์

จุดเด่นของวงโคจรคือการนำเสนอเซ็นเซอร์สุขภาพที่หลากหลายในราคาที่เข้าถึงได้ ไม่ว่าจะเป็นการวัดชีพจร ระดับออกซิเจนในเลือด การติดตามการนอนหลับ หรือโหมดออกกำลังกายที่หลากหลาย สินค้าบางรุ่นออกแบบสำหรับกลุ่มเฉพาะ เช่น เด็ก หรือนักวิ่ง

### จุดเชื่อม (JudChuem) — อุปกรณ์เสริมและการเชื่อมต่อ

จุดเชื่อมเป็นแบรนด์อุปกรณ์เสริมครบวงจร ครอบคลุมสายชาร์จ, ชาร์จเจอร์, แท่นชาร์จไร้สาย, เคสโทรศัพท์, ฟิล์มกันรอย, USB Hub, Dock สำหรับแล็ปท็อป และอุปกรณ์เสริมอีกหลายประเภท

แบรนด์นี้ถูกสร้างขึ้นเพื่อตอบโจทย์ผู้ใช้ที่ต้องการอุปกรณ์เสริมคุณภาพดีในราคาที่สมเหตุสมผล โดยสินค้าทุกชิ้นผ่านการทดสอบความเข้ากันได้กับสินค้าในเครือฟ้าใหม่ และยังรองรับอุปกรณ์มาตรฐานสากลอย่าง USB-C, Qi Wireless Charging และ Thunderbolt อีกด้วย

## แบรนด์พันธมิตร (Partner Brands)

นอกจากสินค้าในเครือ ฟ้าใหม่ยังคัดสรรแบรนด์พันธมิตรที่มีคุณภาพมาจำหน่ายในแพลตฟอร์ม โดยมีเกณฑ์การคัดเลือกที่เข้มงวดเพื่อให้มั่นใจว่าลูกค้าจะได้รับสินค้าที่คุ้มค่าและมีการรับประกันที่ชัดเจน

### ZenByte
แบรนด์เฉพาะทางด้านอุปกรณ์จัดเก็บข้อมูลและหน่วยความจำ ZenByte เป็นที่รู้จักในหมู่ผู้ใช้ที่ต้องการอุปกรณ์จัดเก็บข้อมูลที่เชื่อถือได้และทนทาน สินค้าของ ZenByte จำหน่ายบนฟ้าใหม่ครอบคลุมทั้ง SSD และ RAM สำหรับแล็ปท็อป โดยเฉพาะรุ่นที่รองรับการอัปเกรดได้

### NovaTech
NovaTech เป็นแบรนด์เทคโนโลยีที่มีสินค้าหลากหลาย ตั้งแต่อุปกรณ์คอมพิวเตอร์รอบข้างอย่างคีย์บอร์ดและเมาส์ ไปจนถึงแล็ปท็อปอัลตราบุ๊กและหูฟัง True Wireless สินค้าของ NovaTech มีจุดเด่นด้านการออกแบบที่เรียบร้อยและคุณภาพการผลิตที่ดี

### PulseGear
PulseGear เชี่ยวชาญด้านอุปกรณ์ชาร์จและพลังงาน ไม่ว่าจะเป็นพาวเวอร์แบงก์ความจุสูงหรือแท่นชาร์จไร้สาย สินค้าของ PulseGear โดดเด่นในเรื่องของการขยายประกันเพิ่มเติมได้เมื่อลงทะเบียนออนไลน์

### ArcWave
ArcWave นำเสนอสินค้าในกลุ่มจอมอนิเตอร์และระบบเสียงสำหรับบ้าน โดยมุ่งเน้นผู้ใช้ที่ต้องการประสบการณ์มัลติมีเดียในบ้านที่สมบูรณ์แบบ สินค้าของ ArcWave บนฟ้าใหม่ครอบคลุมทั้งหน้าจอสำหรับทำงานและระบบเสียง Soundbar

## FahMai Membership Program

ฟ้าใหม่มีโปรแกรมสมาชิกที่ให้สิทธิประโยชน์หลากหลายแก่ลูกค้าที่ช้อปปิ้งกับเราอย่างต่อเนื่อง สมาชิกจะได้รับ FahMai Points จากการซื้อสินค้า และสามารถนำ Points ไปแลกเป็นส่วนลดในการซื้อครั้งต่อๆ ไปได้ ระบบสมาชิกมีหลายระดับ โดยระดับที่สูงขึ้นจะได้รับสิทธิพิเศษเพิ่มเติม เช่น จัดส่งฟรี บริการเคลมด่วน และโปรโมชันพิเศษ
