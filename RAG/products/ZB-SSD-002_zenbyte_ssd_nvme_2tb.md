# เซนไบต์ SSD NVMe 2TB (ZenByte SSD NVMe 2TB)

รหัสสินค้า: ZB-SSD-002
แบรนด์: เซนไบต์ (ZenByte) — แบรนด์พันธมิตร
หมวดหมู่: อุปกรณ์จัดเก็บข้อมูล
ราคา: ฿5,990
สถานะ: มีสินค้า
วันที่อัปเดต: 1 มีนาคม 2569

## รายละเอียดสินค้า

สำหรับผู้ที่พื้นที่จัดเก็บข้อมูลไม่เคยพอ ไม่ว่าจะเป็นนักตัดต่อวิดีโอ, นักพัฒนาซอฟต์แวร์ที่รัน virtual machine หลายตัว หรือเกมเมอร์ที่สะสมเกมไว้เต็มไดรฟ์ ZenByte SSD NVMe 2TB คือคำตอบ ด้วยความจุ 2TB ในฟอร์มแฟกเตอร์ M.2 2280 น้ำหนักไม่ถึง 10 กรัม ความจุมหาศาลนี้ไม่ได้มาพร้อมความยุ่งยากด้านพื้นที่และน้ำหนักแต่อย่างใด

ประสิทธิภาพของรุ่น 2TB ยังคงอยู่ในระดับเดียวกันกับรุ่น 1TB ด้วยความเร็วอ่านสูงสุด 7,000 MB/s ผ่านอินเทอร์เฟซ PCIe 4.0 x4 แต่ความเร็วเขียนปรับขึ้นมาที่ 6,000 MB/s ได้ประโยชน์จากการที่ไดรฟ์มี NAND chip มากขึ้น ทำให้การเขียนข้อมูลขนาดใหญ่ทำได้เร็วและต่อเนื่องกว่า ค่าความทนทาน TBW ที่ 1,200 TBW เหมาะกับงานที่ต้องเขียนข้อมูลซ้ำเป็นประจำโดยไม่ต้องกังวลเรื่องอายุการใช้งาน

## สเปคสินค้า

| รายการ | รายละเอียด |
|--------|-----------|
| ความจุ | 2TB (2,000GB) |
| อินเทอร์เฟซ | PCIe 4.0 x4, NVMe 1.4 |
| ฟอร์มแฟกเตอร์ | M.2 2280 |
| ความเร็วอ่านข้อมูล (Sequential Read) | สูงสุด 7,000 MB/s |
| ความเร็วเขียนข้อมูล (Sequential Write) | สูงสุด 6,000 MB/s |
| IOPS อ่าน (Random Read 4K) | สูงสุด 750,000 IOPS |
| IOPS เขียน (Random Write 4K) | สูงสุด 700,000 IOPS |
| ความทนทาน (TBW) | 1,200 TBW |
| ประเภทหน่วยความจำแฟลช | 3D TLC NAND |
| Power Loss Protection | มี |
| น้ำหนัก | ~9 กรัม |
| ขนาด | 80 × 22 × 3.5 มม. |

## สิ่งที่อยู่ในกล่อง

- ZenByte SSD NVMe 2TB ×1
- แผ่นนำความร้อน (Thermal pad) ×1
- คู่มือการติดตั้ง (ภาษาไทย/English)

*หมายเหตุ: ไม่รวมสกรูยึด M.2 (ใช้สกรูที่มาพร้อมเมนบอร์ดของท่าน)*

## การรับประกัน

ZenByte SSD NVMe 2TB มีประกันสินค้า **5 ปี** นับจากวันที่ซื้อ โดยครอบคลุมความบกพร่องจากกระบวนการผลิตและความเสียหายของหน่วยความจำภายใต้การใช้งานปกติ

**สิ่งที่ประกันครอบคลุม:**
- ความบกพร่องจากการผลิต
- ความเสียหายของชิปหน่วยความจำโดยไม่ได้เกิดจากการใช้งานผิดวิธี

**สิ่งที่ไม่ครอบคลุม:**
- ความเสียหายจากอุบัติเหตุ เช่น แรงกระแทก ไฟฟ้าเกิน ไฟฟ้าสถิต
- การดัดแปลงหรือซ่อมแซมโดยบุคคลที่ไม่ได้รับอนุญาต
- ค่า TBW เกินขีดจำกัดที่ระบุ (1,200 TBW)
- การสึกหรอตามปกติของหน่วยความจำแฟลชตามอายุการใช้งาน
