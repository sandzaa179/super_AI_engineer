# พัลส์เกียร์ ChargePad 15W (PulseGear ChargePad 15W)

รหัสสินค้า: PG-CH-001
แบรนด์: พัลส์เกียร์ (PulseGear) — แบรนด์พันธมิตร
หมวดหมู่: แท่นชาร์จไร้สาย
ราคา: ฿890
สถานะ: มีสินค้า
วันที่อัปเดต: 1 มีนาคม 2569

## รายละเอียดสินค้า

PulseGear ChargePad 15W คือแท่นชาร์จไร้สายมาตรฐาน Qi ที่วางแล้วชาร์จได้ทันที รองรับกำลังไฟสูงสุด 15W สำหรับสมาร์ทโฟนที่รองรับ Fast Wireless Charging และ 5W-10W สำหรับอุปกรณ์ Qi ทั่วไป ไฟ LED แสดงสถานะที่ขอบแท่นบอกให้รู้ว่าอุปกรณ์วางถูกตำแหน่งและกำลังชาร์จอยู่ ตัวแท่นบางเรียบ วางบนโต๊ะได้เรียบร้อย ไม่กินพื้นที่

## สเปคสินค้า

| รายการ | รายละเอียด |
|--------|-----------|
| กำลังไฟ Output สูงสุด | 15W (Qi Fast Wireless) |
| กำลังไฟ Output มาตรฐาน | 5W / 7.5W / 10W / 15W (ขึ้นกับอุปกรณ์) |
| มาตรฐาน | Qi (ทุกอุปกรณ์ที่รองรับ Qi) |
| Input | USB-C 5V/3A (15W) |
| ไฟแสดงสถานะ | LED (สีน้ำเงิน = กำลังชาร์จ, สีแดง = ตำแหน่งผิด) |
| ขนาดแท่น | 100 × 100 × 7 มม. |
| น้ำหนัก | 80 กรัม |
| วัสดุพื้นผิว | ยางซิลิโคนกันลื่น |
| สี | สีดำ (Matte Black) |

## สิ่งที่อยู่ในกล่อง

- PulseGear ChargePad 15W ×1
- สาย USB-C 1m ×1
- คู่มือการใช้งาน (ภาษาไทย/English)

*หมายเหตุ: ไม่รวมหัวชาร์จ USB-C แนะนำใช้หัวชาร์จ 15W ขึ้นไปเพื่อให้ได้ความเร็วสูงสุด*

## การรับประกัน

PulseGear ChargePad 15W มีประกันสินค้า **1 ปี** นับจากวันที่ซื้อ

**ขยายประกันเป็น 2 ปีได้ฟรี:** ลงทะเบียนสินค้าออนไลน์ที่ **pulsegear.co.th** ภายใน 30 วันหลังซื้อ เพื่อขยายระยะเวลาประกันจาก 1 ปีเป็น 2 ปีโดยไม่มีค่าใช้จ่ายเพิ่มเติม

**สิ่งที่ไม่ครอบคลุม:**
- ความเสียหายจากการใช้หัวชาร์จที่ไม่ได้มาตรฐาน
- ความเสียหายจากน้ำหรือของเหลว
- การดัดแปลงหรือซ่อมแซมโดยไม่ได้รับอนุญาต

ติดต่อศูนย์บริการ PulseGear: support.pulsegear.co.th

## ความเข้ากันได้

ChargePad 15W รองรับอุปกรณ์ทุกรุ่นที่มีมาตรฐาน Qi รวมถึงสมาร์ทโฟนสายฟ้า (SaiFah) ทุกรุ่นที่รองรับการชาร์จไร้สาย และสมาร์ทวอทช์วงโคจรที่มีฐานชาร์จแบบ Qi

## คำถามที่พบบ่อยเกี่ยวกับสินค้านี้

**ถาม: ใช้กับ สายฟ้า โฟน X9 Pro ได้ไหมครับ?**
ตอบ: ได้เลยครับ สายฟ้า X9 Pro รองรับ Qi มาตรฐาน 15W อยู่แล้ว ChargePad 15W ชาร์จได้เต็มความเร็วครับ
