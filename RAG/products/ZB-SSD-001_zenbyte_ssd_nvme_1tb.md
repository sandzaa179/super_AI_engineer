# เซนไบต์ SSD NVMe 1TB (ZenByte SSD NVMe 1TB)

รหัสสินค้า: ZB-SSD-001
แบรนด์: เซนไบต์ (ZenByte) — แบรนด์พันธมิตร
หมวดหมู่: อุปกรณ์จัดเก็บข้อมูล
ราคา: ฿3,490
สถานะ: มีสินค้า
วันที่อัปเดต: 1 มีนาคม 2569

## รายละเอียดสินค้า

เซนไบต์ SSD NVMe 1TB คือตัวเลือกอัปเกรดที่คุ้มค่าสำหรับผู้ที่ต้องการเพิ่มความเร็วและความจุให้เครื่องคอมพิวเตอร์ของตัวเอง ด้วยอินเทอร์เฟซ PCIe 4.0 และความเร็วอ่านข้อมูลสูงถึง 7,000 MB/s และเขียนที่ 5,500 MB/s ทำให้ทุกการโหลดโปรแกรม เปิดไฟล์ขนาดใหญ่ หรือบูตเข้าระบบ รวดเร็วกว่า SSD รุ่นเก่าได้อย่างเห็นได้ชัด

ออกแบบในมาตรฐาน M.2 2280 ที่รองรับช่องเสียบ PCIe 4.0 x4 NVMe บนเมนบอร์ดรุ่นใหม่ทั้งในเดสก์ท็อปและแล็ปท็อปที่มีช่อง M.2 ว่าง ZenByte ใช้วงจรควบคุมที่มีฟีเจอร์ป้องกันข้อมูลเมื่อไฟฟ้าดับ (Power Loss Protection) และระบบแก้ไขข้อผิดพลาด (ECC) เพื่อให้ข้อมูลของคุณปลอดภัยตลอดอายุการใช้งาน ด้วยค่าความทนทาน TBW 600TB ความจุ 1TB นี้จึงรองรับการใช้งานหนักได้อีกนานหลายปี

## สเปคสินค้า

| รายการ | รายละเอียด |
|--------|-----------|
| ความจุ | 1TB (1,000GB) |
| อินเทอร์เฟซ | PCIe 4.0 x4, NVMe 1.4 |
| ฟอร์มแฟกเตอร์ | M.2 2280 |
| ความเร็วอ่านข้อมูล (Sequential Read) | สูงสุด 7,000 MB/s |
| ความเร็วเขียนข้อมูล (Sequential Write) | สูงสุด 5,500 MB/s |
| IOPS อ่าน (Random Read 4K) | สูงสุด 700,000 IOPS |
| IOPS เขียน (Random Write 4K) | สูงสุด 650,000 IOPS |
| ความทนทาน (TBW) | 600 TBW |
| ประเภทหน่วยความจำแฟลช | 3D TLC NAND |
| Power Loss Protection | มี |
| น้ำหนัก | ~8 กรัม |
| ขนาด | 80 × 22 × 3.5 มม. |

## สิ่งที่อยู่ในกล่อง

- ZenByte SSD NVMe 1TB ×1
- แผ่นนำความร้อน (Thermal pad) ×1
- คู่มือการติดตั้ง (ภาษาไทย/English)

*หมายเหตุ: ไม่รวมสกรูยึด M.2 (ใช้สกรูที่มาพร้อมเมนบอร์ดของท่าน)*

## การรับประกัน

ZenByte SSD NVMe 1TB มีประกันสินค้า **5 ปี** นับจากวันที่ซื้อ โดยครอบคลุมความบกพร่องจากกระบวนการผลิตและความเสียหายของหน่วยความจำภายใต้การใช้งานปกติ

**สิ่งที่ประกันครอบคลุม:**
- ความบกพร่องจากการผลิต
- ความเสียหายของชิปหน่วยความจำโดยไม่ได้เกิดจากการใช้งานผิดวิธี

**สิ่งที่ไม่ครอบคลุม:**
- ความเสียหายจากอุบัติเหตุ เช่น แรงกระแทก ไฟฟ้าเกิน ไฟฟ้าสถิต
- การดัดแปลงหรือซ่อมแซมโดยบุคคลที่ไม่ได้รับอนุญาต
- ค่า TBW เกินขีดจำกัดที่ระบุ (600 TBW)
- การสึกหรอตามปกติของหน่วยความจำแฟลชตามอายุการใช้งาน
